Copy original titus/moktar files to the titus/moktar subfolders. Launch opentitus.exe.

Keys:
Left/Right: Walk
Up: Jump
Down: Kneestand
Down (on certain places): Enter secret areas/drop through the floor
Down + Space: Pick up or put down an object (or enemy)
Space: Throw
Down + left/right: Crouching
Down + Up: Initiate silent walk (will not wake up the dogs,
           makes it easier to pick up the walking men from behind)
F1: Die
F2: Game over
F3: Toggle audio
F4: Status page
E: Health display
P: Pause
Esc: Exit
L: Toggle Amiga lines
C: Display credits in menu

Development mode:
change devmode to 1 in titus.conf to enable development mode
N: Noclip
G: Godmode
S: Skip to next level
